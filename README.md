# Learning-Explainable-AI

Boruta is an all-relevant feature selection algorithm that can automatically identify important features by comparing them with their shuffled versions. It is a wrapper algorithm based on the random forest classifier. Its primary goal is to find all features carrying information usable for prediction. It aims to retain all features that have an impact on the outcome. The Boruta algorithm differs from standard Random Forest (RF) feature selection techniques by its unique approach to identifying all relevant features rather than just a subset. While traditional RF methods may select features based solely on their importance scores, Boruta employs a more comprehensive strategy by creating “shadow features” — randomly permuted copies of the original features.
